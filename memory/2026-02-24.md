# 2026-02-24

## 今日事件
- 完成 nullbr-sse MCP（通过 mcporter）接入与连通验证。
- 新增自然语言触发约定：`搜 <关键词>`，由助手自动映射到 `mcporter call nullbr-sse.search_media query=<关键词> page:1`。
- 新增二段式查询约定：
  - 用户回复 `tmdb_id: <id>` → 默认查询详情（movie_detail / tv_detail）
  - 用户回复 `tmdb_id: <id> 资源 <类型>` → 查询资源（movie_resources / tv_resources）
- 资源优先策略：优先返回 115 资源；若无 115 再考虑 magnet / ed2k / video。

## 技术笔记
- mcporter 添加 SSE 服务：
  - `mcporter config add nullbr-sse "https://mcp.nullbr.com/sse?api_key=<KEY>"`
- 工具发现：
  - `mcporter list nullbr-sse --schema`
- 搜索：
  - `mcporter call nullbr-sse.search_media query=<关键词> page:1`
- 电影详情：
  - `mcporter call nullbr-sse.movie_detail tmdb_id:<id>`
- 电影资源（注意 resource_type=115 建议用 JSON args 以避免类型解析问题）：
  - `mcporter call nullbr-sse.movie_resources --args '{"tmdb_id":1726,"resource_type":"115","page":1}'`

# OpenClaw 系统目录结构（当前）

```text
~/.openclaw/workspace/
├── AGENTS.md                # 工作区总规则（行为、记忆、心跳）
├── SOUL.md                  # 助手人格与语气
├── USER.md                  # 用户信息与偏好
├── MEMORY.md                # 长期记忆（精华）
├── HEARTBEAT.md             # 心跳任务清单
├── BOOTSTRAP.md             # 初始引导说明（首次配置后可删除）
├── IDENTITY.md              # 助手身份设定
├── TOOLS.md                 # 本地工具/环境备忘
│
├── memory/                  # 每日原始日志
│   ├── TEMPLATE.md          # 每日日志模板
│   ├── 2026-02-12.md
│   └── 2026-02-12-0038.md
│
├── learnings/               # 学习总结（提炼版）
│   └── README.md
│
├── skills/                  # 本地安装的 skills（目前为空）
└── .git/                    # Git 版本管理目录
```

## 建议后续补充
- `learnings/YYYY-MM-DD.md`：每天总结输出
- `scripts/`：常用自动化脚本（例如 pushplus 推送、检查脚本）
- `docs/`：部署说明、迁移说明、排错手册
